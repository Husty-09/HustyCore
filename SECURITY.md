# Security Policy

The security of **HustyCore** is taken very seriously. If you discover a security vulnerability in this framework, I appreciate your help in reporting it in a responsible and confidential manner.

## Supported Versions

Currently, only the latest versions receive security updates and patches.

| Version | Security Updates |
| ------- | ---------------- |
| 1.0.x   | :white_check_mark: Yes |
| < 1.0   | :x: No           |

## How to Report a Vulnerability

**Please do not report security vulnerabilities through public GitHub *Issues*.**

Instead, we ask that you send the details of the vulnerability directly by email to: **matheusscalonico@gmail.com**.

In your report, please try to include as much information as possible to help us understand and resolve the issue quickly:
* A summary of the vulnerability (e.g., XSS in the rendering of a specific component).
* Detailed steps to reproduce the issue.
* The potential impact of the vulnerability.

### Response
You should receive a confirmation of receipt of your email. If the issue is confirmed, i will work to release a patch as quickly as possible prior to any public disclosure.
